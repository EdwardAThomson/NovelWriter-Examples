# NovelWriter Examples

This repository contains example outputs generated by [NovelWriter](https://github.com/EdwardAThomson/NovelWriter), an open-source Python application that uses Large Language Models (LLMs) to assist in writing novels and short stories.

## About NovelWriter

NovelWriter is a GUI-based application that breaks down the complex task of novel writing into manageable, automated steps. It leverages various LLMs (OpenAI GPT models, Google Gemini, Anthropic Claude) to generate coherent long-form fiction by following a structured workflow:

1. **Novel Parameters** - Define genre, themes, structure, and basic story elements
2. **Generate Lore** - Create universe background, characters, factions, and world-building
3. **Story Structure** - Develop detailed plot outlines following established narrative frameworks
4. **Scene Planning** - Break down chapters into detailed scene-by-scene plans
5. **Write Chapters** - Generate prose scene-by-scene with full context awareness

For more details about the application, visit the main repository: [NovelWriter](https://github.com/EdwardAThomson/NovelWriter)

## Repository Purpose

This repository serves as a showcase of NovelWriter's capabilities, demonstrating:

- **Real examples** of generated novels and short stories
- **Complete workflow outputs** showing all intermediate generation steps
- **Evolution over time** as the application improves and new features are added
- **Different LLM comparisons** when the same story is generated with different models

## File Structure

Each novel/story project typically contains:

```
Project_Name/
├── combined_novel.md              # Complete novel in single file
├── chapter_1.md                   # Individual chapter files
├── chapter_2.md
├── ...
├── story_outline_files/
│   ├── parameters.txt             # Initial story parameters
│   ├── generated_lore.md          # Universe background
│   ├── characters.md              # Character profiles
│   ├── characters_enhanced.md     # Expanded character profiles
│   ├── factions.md                # Faction details
│   ├── planets.md                 # World-building elements
│   ├── technology.md              # Setting technology
│   ├── relationships.md           # Character relationships
│   ├── story_structure.md         # Plot outline
│   └── chapter_outlines.md        # Chapter summaries
└── scene_outlines/                # Detailed scene plans
    ├── scenes_chapter_1.md
    ├── scenes_chapter_2.md
    └── ...
```

## Key Features Demonstrated

- **Coherent Long-Form Storytelling:** Examples show how the application maintains character consistency, plot continuity, and thematic coherence across thousands of words
- **Rich World-Building:** Generated universes with detailed factions, planets, technologies, and character relationships
- **Structured Approach:** Clear progression from high-level concepts to detailed prose

## Generated Content Quality

These examples represent the current capabilities of LLM-assisted creative writing. While the generated content serves as excellent first drafts and demonstrates the potential of AI-assisted authoring, they may benefit from human editing and refinement for publication-quality work.

## Technical Notes

- All examples were generated using the workflow described in the [NovelWriter documentation](https://github.com/EdwardAThomson/NovelWriter/tree/main/docs)
- Generation requires an API key for the chosen LLM provider
- Total generation time varies but novels typically ranges from 30 minutes to 2 hours depending on story length and model speed. Short stories should be under 10 minutes (across a number of prompts)
- Costs vary by LLM model and story length (typically $5-20 per novel)
